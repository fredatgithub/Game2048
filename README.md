## Description
<!-- Explication générale du projet -->
Une implémentation du jeu 2048 en C# utilisant WPF (Windows Presentation Foundation).
Le but est de combiner des tuiles pour atteindre le nombre 2048.

## Prérequis
<!-- Configuration système et dépendances nécessaires -->
- .NET Framework
- Windows pour exécuter l'application WPF

## Installation
<!-- Instructions pour installer et exécuter le jeu -->
1. Télécharger ou cloner le repository
2. Ouvrir `Game2048.sln` avec Visual Studio
3. Compiler la solution
4. Exécuter l'application

## Fonctionnalités
<!-- Caractéristiques principales du jeu -->
- Interface graphique WPF
- Mécanique de jeu classique du 2048
- Gestion des scores et des mouvements

## Fichiers Principaux
<!-- Description des fichiers clés du projet -->
- `MainWindow.xaml` - Interface utilisateur principale
- `MainWindow.xaml.cs` - Logique de la fenêtre principale
- `App.xaml` - Configuration de l'application
- `Game2048.csproj` - Fichier de projet C#
